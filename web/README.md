# Soundscout Web App
The Soundscout Web App gets soundscout data from Exosite's OnePlatform and displays it both in table and map forms.

## Installation
mkvirtualenv ssw --python=`which python2.7`
pip install -r requirements.txt

## Run the web app
./ssweb.py

## View the data
127.0.0.1:5000
